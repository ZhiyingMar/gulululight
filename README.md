# 留言板项目说明

## 项目简介

项目的功能是做一个基础的留言板功能，可以自己创建账号，用账号发表留言板的相关内容

## 环境部署

### NodeJS

[NodeJS安装](https://nodejs.org/zh-cn)

`npm install`

## 项目启动

`npm start`

## 项目展示地址

 🌐 <https://zhiyingmar.github.io/gulululight/>
